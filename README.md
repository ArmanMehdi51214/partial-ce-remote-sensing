# Partial CE Remote Sensing Segmentation

Implementation of sparse point-supervised semantic segmentation using Partial Cross Entropy Loss.

## Objective
Train a semantic segmentation model using sparse point labels instead of full segmentation masks.

## Features
- Partial Cross Entropy Loss
- Sparse point supervision
- Remote sensing segmentation
- Experimental analysis

## Structure
- notebooks/ : training notebook
- outputs/ : predictions and experiment results
- reports/ : technical report
